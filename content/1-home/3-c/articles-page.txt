    Code is kortgezegd de taal die de computer snapt. Een voorbeeld: een computerprogramma is een opeenvolging van instructies aan de computer om deze iets te laten doen. Als de computer een plaatje op het beeldscherm moet laten zien, wordt hem eigenlijk één voor één verteld welke pixel welke kleur moeten krijgen. Het computerprogramma is geschreven in code, de taal die de computer snapt. Het schrijven van code noemen we (link: home/p#programmeren text: programmeren).
    
    Gelukkig is er al heel veel code door andere mensen geschreven die je op een handige manier kunt gebruiken als je zelf code gaat schrijven. Zo hoef jij uiteindelijk wanneer je een foto op het scherm wilt toveren geen miljoen instructies te geven (voor elke pixel 1), maar ziet jouw code er bijvoorbeeld zo uit:
    LaatPlaatjeZien(“Konijn.jpg”).
    
    Een app is ook een computer programma dat door een programmeur geschreven is in code. De code zorgt er voor dat de computer doet wat de programmeur bedacht heeft op het moment dat jij bijvoorbeeld op een knopje drukt.
    
    Op (link: http://www.codekinderen.nl text: Codekinderen) en (link: http://www.codeuur.nl text: CodeUur) is lesmateriaal beschikbaar om met programmeren aan de slag te gaan. En op (link: http://www.codestarter.nl text: Codestarter) vind je programmeerclubs bij jou in de buurt. Het boek CodeKlas is een inspiratieboek over programmeren met kinderen, voor leraren en ouders van kinderen in het (link: http://www.boektweepuntnul.nl/codeklas/?v=796834e7a283 text: basis- en voortgezet onderwijs).
